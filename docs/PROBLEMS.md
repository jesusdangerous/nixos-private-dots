## Known Problems

Краткий список известных проблем и обходов.

1. Конфликты при ребилде сервисов
- Симптом: зависание или черный экран после switch/test.
- Обход: использовать boot и перезагрузку, затем найти конфликтный сервис.

2. Приложения могут перезаписывать часть пользовательских настроек
- Симптом: часть файлов в ~/.config меняется после запуска GUI приложения.
- Обход: по возможности держать критичные настройки декларативно через Home Manager.

3. TUN/прокси клиенты
- Симптом: нестабильная работа некоторых TUN режимов.
- Обход: проверить права, настройки клиента и версию ядра/системы.

4. Обновления после больших изменений
- Рекомендация: сначала build/test, потом switch.
- Для безопасного процесса см. docs/SETUP_SAFE.md.
