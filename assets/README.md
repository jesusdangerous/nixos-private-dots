Этот каталог хранит статические ассеты для UI:
- цветовые схемы qt5ct/qt6ct
- темы Kvantum
- заготовки некоторых desktop-конфигов

Основной способ применения: декларативно через Home Manager модуль `modules/home-manager/gui/qt.nix`.

Ручной скрипт копирования больше не используется, чтобы структура оставалась предсказуемой и без дублирования.
